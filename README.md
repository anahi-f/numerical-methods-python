# 🧮 Numerical Methods Application

A desktop application developed in **Python** that allows users to solve different **numerical methods** through an interactive graphical interface.

This project was created as part of my studies in **Biomedical Engineering**, implementing numerical algorithms commonly used in scientific and engineering computations.

## 💻 Graphical Interface

The application includes an interactive graphical interface built with **Tkinter**, allowing users to select different numerical methods and provide the necessary input values for each algorithm.

The interface guides the user through the required steps depending on the selected method and displays the results numerically and graphically when applicable.

---

## 🔢 Implemented Numerical Methods

### Linear Systems
- Gaussian elimination (without pivoting)
- Gaussian elimination (with pivoting)
- Gauss-Seidel method

### Nonlinear Equations
- Bisection method
- Newton–Raphson method
- Secant method

### Regression
- Linear regression
- Polynomial regression

### Interpolation
- Lagrange interpolation
- Spline interpolation  
  - Linear spline  
  - Quadratic spline  
  - Cubic spline  

### Numerical Integration
- Trapezoidal rule
- Simpson 1/3 rule
- Simpson 3/8 rule

### Numerical Differentiation
- Forward difference
- Backward difference
- Centered difference
- Richardson extrapolation

### Differential Equations
- Runge–Kutta method

- Interactive graphical interface
- Input validation for numerical data
- Visualization of functions and results
- Modular implementation of numerical algorithms
- Educational tool for learning numerical methods

# 🧮 Aplicación de Métodos Numéricos

Una aplicación de escritorio desarrollada en **Python** que permite resolver diferentes **métodos numéricos** mediante una interfaz gráfica interactiva.

Este proyecto fue desarrollado como parte de mis estudios en **Ingeniería Biomédica**, implementando algoritmos numéricos utilizados en problemas científicos y de ingeniería.

## 💻 Interfaz Gráfica

La aplicación incluye una interfaz gráfica interactiva desarrollada con **Tkinter**, que permite seleccionar distintos métodos numéricos e ingresar los datos necesarios para ejecutar cada algoritmo.

La interfaz guía al usuario a través de los pasos requeridos según el método seleccionado y muestra los resultados de forma numérica y gráfica cuando corresponde.

---

### Sistemas de ecuaciones lineales
- Eliminación de Gauss (sin pivoteo)
- Eliminación de Gauss (con pivoteo)
- Método de Gauss-Seidel

### Ecuaciones no lineales
- Método de bisección
- Método de Newton–Raphson
- Método de la secante

### Regresión
- Regresión lineal
- Regresión polinómica

### Interpolación
- Interpolación de Lagrange
- Interpolación por splines  
  - Spline lineal  
  - Spline cuadrático  
  - Spline cúbico  

### Integración numérica
- Regla del trapecio
- Regla de Simpson 1/3
- Regla de Simpson 3/8

### Diferenciación numérica
- Diferencia hacia adelante
- Diferencia hacia atrás
- Diferencia centrada
- Extrapolación de Richardson

### Ecuaciones diferenciales
- Método de Runge-Kutta
